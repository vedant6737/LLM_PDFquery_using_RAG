 Tech Stack
* Python
* LangChain
* PyPDF2
* HuggingFace Embeddings (all-MiniLM-L6-v2)
* Astra DB (DataStax)
* Apache Cassandra
* Groq
* Llama 3.1

* PDF → Text Extraction → Chunking → Embeddings
→ Astra DB → Vector Search → Llama 3.1 → Answer

Built a PDF-based RAG chatbot using LangChain, HuggingFace Embeddings, Astra DB, and Llama 3.1 to enable semantic document search and context-aware question answering.
